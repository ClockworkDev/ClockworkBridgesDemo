This is a demo of the UWP Clockwork bridge!
Select a .cw file:

Once you click submit, wait a minute while the .appx package is being generated in the server :)

To try the app, first you will need to install this certificate. Download it, right click it and select "install certificate". A wizard will appear, where you have to select "Local machine", click next, then select "Place all certificates in the following store", click browse and select "Trusted People". Click next and finish.

Finally, add ".appx" to the name of the downloaded file, and double click it to install it.
